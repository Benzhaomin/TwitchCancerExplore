Twitch No Cancerino
--------> Firefox Store Chrome Store

A simple browser extension that removes emoticons and hides cancer messages in Twitch's chat.


Usage

Once installed, right-click inside the chatbox and toggle cancer off.

Bad quality messages have their text turned partly transparent to help you read good quality messages easily and ignore bad ones.

Chat messages from regular Twitch users (plebs) are checked for signs of bad quality posting like repeated words, lots of capital letters, lots of emotes, a high ressemblance to other long messages (copy-pasta), and more.

This only affects the current chat and resets to off when leaving or refreshing the page.

Questions

I see empty messages from subs
All emoticons are removed but we also always let sub messages through which can lead to empty messages.
A workaround is to install a user-side CSS extension add a rule like:
.sane img { display: block !important; }
